# Super Toolbox
Super toolbox is a offline devtools suite in a compact html with fast loading times.

Single‑file offline developer toolbox with 40+ utilities.

## Features
- Base64, URL, HTML encode/decode  
- SHA‑1, SHA‑256, MD5, AES‑GCM, JWT  
- CSV/JSON/YAML converters  
- Markdown preview, sandbox  
- HTML/CSS/JS minifiers  
- Regex tester + explainer  
- Diff viewer, text stats  
- QR, UUID, random tools  
- ASCII, noise, password strength  
- Terminal, JS REPL, pixel canvas  

## Usage
Open `super-toolbox.html` in any browser.  
Everything runs offline.  

## Structure
- Readable HTML  
- Compact JS block  
- No dependencies  

## Theme
Dark/light toggle with localStorage.
